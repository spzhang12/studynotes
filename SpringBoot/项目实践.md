# Spring Boot项目实践

**手机商城**

基于移动端的手机商城

基于Spring Boot + Vue的前后端分离开发架构

**技术栈**
前端技术栈：Vue + Vant UI + less

后端技术栈：Spring Boot + SpringDataJPA

数据库：MySQL

## 1. Vue概述

前后端开发者只需要提前约定好接口文档（URL、参数、数据类型等），分别独立开发即可。前端可以造假数据进行测试，完全不需要依赖后端，最后完成前后端集成即可，真正实现了前后端应用的解耦合，极大地提升了开发效率。

单体 --> 前端应用 + 后端应用

前端应用：负责数据展示和用户交互

后端应用：负责提供数据处理接口

前端HTML --> Ajax --> RESTful后端接口



前后端分离就是将一个应用拆分成两个独立的应用，前端应用和后端应用以JSON格式进行数据交互。

**实现技术**

Spring Boot + Vue
