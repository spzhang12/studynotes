### list类型
- 存储多个string数据，并对数据进入存储空间的顺序进行区分
- 底层使用双向链表存储结构实现

#### 基本操作
添加/修改数据
```
lpush key value1 [value2]   ...         //插入到链表的头部
rpush key value1 [value2]   ...         //插入到链表尾部
```

获取数据
```
lrange key start stop
lindex key index
llen key
```

获取并移除数据
```
lpop key            //从链首获取数据并将其删除
rpop key            //从链尾获取数据并将其删除
```
