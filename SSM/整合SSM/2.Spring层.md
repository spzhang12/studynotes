# 2. Spring层

整合dao层:spring-dao.xml

1. 关联数据库配置文件

   ```xml
    <!-- 导入数据库配置文件 -->
   <context:property-placeholder location="classpath:dp.properties" />
   ```

2. 连接池

   - dbcp：半自动化，不能自动连接

   - c3p0：自动话操作（自动话的加载配置文件，并且可以自动设置到对象中！）
     
     - 设置私有属性：maxPoolSize、minPoolSize、autoCommitOnClose、checkoutTimeout
     
   - **druid**

   - hikari

     ```xml
     <!-- 配置连接池，这里选择c3p0 -->
     <bean id="dataSource" class="com.mchange.v2.c3p0.ComboPooledDataSource">
         <property name="driverClass" value="${jdbc.driver}" />
         <property name="jdbcUrl" value="${jdbc.url}" />
         <property name="user" value="${jdbc.username}" />
         <property name="password" value="${jdbc.password}" />
     
         <!-- c3p0连接的私有属性 -->
         <property name="maxPoolSize" value="30" />
         <property name="minPoolSize" value="10" />
         <property name="autoCommitOnClose" value="false" />
         <property name="checkoutTimeout" value="10000" />
         <property name="acquireRetryAttempts" value="2" />
     </bean>
     ```

3. SqlSessionFactory

   - 注入数据源
   - 绑定mybatis核心配置文件

   ```xml
   <!-- 配置sqlSessionFactory -->
   <bean class="org.mybatis.spring.SqlSessionFactoryBean" id="sqlSessionFactory">
       <property name="dataSource" ref="dataSource" />
       <property name="configLocation" value="classpath:mybatis-config.xml" />
   </bean>
   ```

4. 配置dao接口扫描包

   - 注入sqlSessionFactory
   - 配置要扫描的包路径

   ```xml
   <!-- 配置dao接口扫描包 -->
   <bean class="org.mybatis.spring.mapper.MapperScannerConfigurer">
       <!-- 注入sqlSessionFactory，使用value -->
       <property name="sqlSessionFactoryBeanName" value="sqlSessionFactory" />
       <property name="basePackage" value="com.spzhang.dao" />
   </bean>
   ```

整合service层spring-service.xml

1. 创建spring-service.xml

2. 将所有业务类注入到spring，可以通过配置或者开启注解

   - 配置

     ```xml
     <!-- 将所有的业务类注入到Spring中，可以通过配置，也可以通过注解 -->
     <!-- 配置 -->
     <bean id="bookService" class="com.spzhang.service.BookServiceImpl">
         <property name="bookMapper" ref="bookMapper" />
     </bean>
     ```

   - 注解

   ```xml
   <!--自动扫描及自动注入 -->
   <!-- 扫描service下的包 -->
   <context:component-scan base-package="com.spzhang.service" />
   <context:annotation-config />
   ```

3. 声明式事务配置

   ```xml
   <!-- 声明式事务管理 -->
   <bean id="transactionManager" class="org.springframework.jdbc.datasource.DataSourceTransactionManager">
       <property name="dataSource" ref="dataSource" />
   </bean>
   ```
